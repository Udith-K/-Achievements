# Splunk – The Art of Investigation

This folder contains my learning artifacts from the Splunk eLearning course **The Art of Investigation**.

## Topics Covered

- Splunk SPL investigation workflow
- Log analysis using stats and timechart
- VPN and Salesforce login correlation
- Geolocation analysis using iplocation and geostats
- IOC investigation using threat intelligence tools

## Tools Used

- Splunk Enterprise
- AlienVault OTX
- Shodan
- AbuseIPDB
- VirusTotal

## Files

- Certificate of completion
